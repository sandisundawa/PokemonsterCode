PokemonsterCode ini adalah aplikasi yang memuat data dari pokeAPI, dari test yang diberikan oleh Code.id,
dengan spesifikasi (yang telah diselesaikan) sebagai berikut :
- Data dari https://pokeapi.co/
- Jetpack Compose UI
- MVVM, with DDD architecture
- Retrofit
- Coroutine
- Room local Database
- Offline Capability

Screen :
- Registrasi
- Login
- Landing : Home, Profile
- Detail Pokemon
